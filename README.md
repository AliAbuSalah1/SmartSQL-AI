# SmartSQL AI

AI-powered Natural Language to SQL Assistant.

## Description

SmartSQL AI allows users to ask questions using natural language and converts them into SQL queries using Large Language Models.

## Features

- Natural Language to SQL
- Safe SQL execution
- Database schema understanding
- Query history
- Data visualization

## Tech Stack

### Backend
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy

### Frontend
- React
- Vite

### AI
- LLM
